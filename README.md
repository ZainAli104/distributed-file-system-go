# Distributed File System in Golang

Features:

- Files can be accessed from any node in the network

### Prerequisites

**Go version 1.23**

### Cloning the repository

```shell
git clone https://github.com/ZainAli104/distributed-file-system-go
```

### Install packages

```shell
go get
```

### Commands to start

Build the project

```shell
make build
```

Run the project
```shell
make run
```

### Testing

```shell
make test
```
